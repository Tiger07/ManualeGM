Modificare i tuoi sprite

Finora abbiamo caricato i nostri sprite da dei file. È anche possibile crearli e modificarli internamente a GameMaker. Per farlo, apri la finestra di uno sprite facendo doppio clic su uno di essi (o creandone uno nuovo). Ora premi il pulsante Edit Sprite (modifica sprite). Apparirà una nuova finestra che mostra tutte le subimmagini che compogono lo sprite.

L'editor di sprite ha quest'aspetto:

Sulla destra puoi vedere le diverse subimmagini che compongono lo sprite. Nota che in GameMaker tutte le subimmagini di uno sprite devono avere le stesse dimensioni. Sulla sinistra puoi vedere l'animazione delle sprite. (Se non la vedi, clicca su Show Preview (mostra anteprima).) Sotto l'anteprima puoi cambiare la velocità dell'animazione e il colore di sfondo. Puoi anche scegliere uno dei background presenti nel gioco e decidere se allungarlo o meno. Così puoi farti un'idea di come uscirà l'animazione nel gioco. (Nota che questa velocità è solo un'anteprima. La velocità dell'animazione durante il gioco dipende dal parametro "room speed", che puoi impostare nella room.)

L'editor di sprite contiene tanti comandi per creare e cambiare lo sprite. Sono tutti accessibili dai menu. (Per alcuni ci sono pulsanti sulla barra degli strumenti.) Alcuni comandi lavorano su singole subimmagini. Richiedono che tu selezioni un'immagine col mouse prima.

Il menu File

Il menu File contiene dei comandi per caricare e salvare gli sprite.

  • New. (Nuovo) Crea un nuovo sprite vuoto. Devi indicarne la dimensione. (Ricorda, tutte le immagini di uno sprite devono avere la stessa dimensione.)
  • Create from File. (Crea da file) Crea uno sprite da un file. Puoi usare molti tipi di file. Verrà creato sempre uno sprite costituito da una singola subimmagine, a parte i GIF animati che sono divisi in subimmagini e le strip che hanno il nome che finisce con _stripXX. Mentre selezioni un file puoi indicare se l'immagine dev'essere opaca, se rimuovere lo sfondo o no, e se smussare gli angoli. Puoi scegliere più immagini che verranno caricate insieme. Dovrebbero avere le stesse dimensioni, altrimenti le loro dimensioni canvas verranno adattate.
  • Add from File. (Aggiungi da file) Aggiunge una o più immagini da un file allo sprite corrente. Se le immagini non hanno la stessa dimensione puoi decidere dove posizionarle o se allungarle. Puoi selezionare più immagini che verranno caricate insieme. Devono avere la stessa dimensione.
  • Save as PNG File. (Salva come un PNG) Salva lo sprite come una strip in un file PNG. Se lo sprite ha più subimmagini il testo _stripXX dove XX è il numro di subimmagini viene aggiunto automaticamente.
  • Create from Strip. (Crea da strip) Ti permette di creare uno sprite da una strip. Vedi la prossima sezione per maggiori informazioni.
  • Add from Strip. (Aggiungi da strip) Aggiunge le immagini da una strip. Vedi la prossima sezione.
  • Close Saving Changes. (Chiudi salvando i cambiamenti) Chiude la finestra, salvando le modifiche apportate. Se non vuoi salvare, clicca sul pulsante "Chiudi" della finestra.

Il menu Edit (modifica)

Il menu Edit contiene i segunti comandi:

  • Undo. (Annulla) Annulla l'ultimo cambiamento. Puoi annullare le prime 16 modifiche.
  • Redo. (Ripeti) Ripete l'ultimo cambiamento annullato.
  • Cut. (Taglia) Taglia l'immagine selezionata e la salva negli appunti. Puoi modificare o usare la subimmagine in un altro progrmma così. Tuttavia, le informazioni sulla trasparenza potrebbero andare perdute.
  • Copy. (Copia) Copia l'immagine selezionata e la salva negli appunti.
  • Paste. (Incolla) Incolla l'immagine negli appunti come una nuova sibimmagine. Se non ha le dimensioni corrette dovrai indicare come cambiarle. Se l'immagine viene da un altro programma, le informazioni sulla trasparenza potrebbero andare perdute.
  • Erase. (Cancella) Cancella una o tutte le subimmagini, riempiendole completamente con un colore. Si apre una finestra in cui puoi scegliere il colore e la trasparenza (alpha). Puoi vedere un'anteprima del risultato. Puoi scegliere se cancellare tutte le subimagini o solo quella corrente.
  • Delete. (Elimina) Elimina la subimmagine selezionata.
  • Move Left. (Muovi a sinistra) Muove la subimmagine selezionata di una posizione a sinistra nella sequenza.
  • Move Right. (Mouvi a destra) Muove la subimmagine selezionata di una posizione a destra nella sequenza.
  • Add Empty. (Aggiungi vuota) Aggiunge una nuova subimmagine vouta alla fine della sequenza.
  • Insert Empty. (Inserisci vuota) Aggiunge una nuova subimmagine vuota prima della subimmagine selezionata.
  • Edit. (Modifica) Modifica la subimmagine selezionata nell'editor di immagini integrato.
  • Set Tranparency Background. (Imposta sfondo di trasparenza) Qui puoi indicare come dev'essere mostrata la parte trasparente dello sprite all'interno dell'editor. Puoi usare un motivo a blocchi personalizzabile, o un colore che puoi cambiare cliccando sulla casella colorata.

Il menu Transform (trasforma)

Nel menu Transform puoi applicare alcune trasformazioni alle subimmagini. Per la maggior parte puoi scegliere se applicarle solo alla subimmagine corrente o a tutte le subimmagini.

  • Shift. (Sposta) Qui puoi spostare la(le) subimmagine(i) di un numero di pixel dato in orizzontale e in verticale.
  • Mirror/Flip. (Immagine speculare/capovolta) Specchia la(le) subimmagine(i) orizzontalmente e/o la(le) capovolge verticalmente.
  • Rotate. (Routa) Puoi ruotare la(le) subimmagine(i) di 90, 180 o 270 gradi, o di un valore personalizzato.
  • Scale. (Ridimensiona) Questo comando ingrandisce o rimpicciolisce la(le) subimmagine(i) (ma non ne cambia la dimensione!). Puoi indicare il fattore di ridimensionamento e se applicarlo orizzontalmente e/o verticalmente.
  • Skew. (Distorci) Questo comando distorce la(le) subimmagine(i) orizzontalmente e/o verticalmente di un valore dato.
  • Resize Canvas. (Ridimensiona canvas) Qui puoi cambiare la dimensione del canvas (lo spazio trasparente su cui si può disegnare) Puoi anche indicare dove le subimmagini verranno posizionate sul nuovo canvas. (Questo comando cambia la dimensione delle subimmagini.)
  • Stretch. (Allarga) Qui puoi allargare le subimmagini cambiandone la dimensione. Puoi indicare il fattore di ridimensionamento e la qualità.
  • Crop. (Ritaglia) Rende le subimmagini più piccole possibili riducendone le dimensioni. È molto utile perché le immagini più grandi consumano una maggiore quantità di memoria video. Puoi indicare la dimensione di un bordo trasparente attorno alle immagini. Se fornisci un valore negativo, la parte esterna delle immagini verrà tagliata via.

Il menu Images (immagini)

Nel menu Images puoi eseguire alcune operazioni sulle immagini. Alcune sono disponibili solo nella Standard Edition.

  • Cycle Left. (Sposta tutto a sinistra) Sposta tutte le subimmagini di un posto a sinistra. Di fatto fa cominciare l'animazione da un punto diverso.
  • Cycle Right. (Sposta tutto a destra) Sposta tutte le subimmagini di un posto a destra.
  • Black and White. (Bianco e nero) Converte la(le) subimmagine(i) in bianco e nero (ovvero in scala di grigi).
  • Colorize. (Tingi) Qui puoi cambiare il colore (tonalità) delle subimmagini. Usa il cursore per prendere una nuova tonalità. Quando scegli di aggiungere tonalità (casella "Shift the hue") ogni colore è cambiato separatamente in base alla posizione del cursore, dando effetti piuttosto interessanti.
  • Colorize Partial. (Tingi parzialmente) Qui puoi cambiare il colore (tonalità) di una parte dell'immagine. Puoi selezionare la vecchia tonalità e il margine (tolerance) attorno a essa (più è alto, più tonalità simili vengono cambiate oltre a quella scelta) e poi indicare la nuova tonalità con cui rimpiazzare quell'insieme di colori. Può essere usato, ad esempio, per cambiare solo il colore della maglietta dei giocatori. Puoi selezionare la vecchia tonalità anche cliccando sull'immagine a sinistra.
  • Intensity. (Intensità) Qui puoi cambiare l'intensità fornendo il valore per la saturazione dei colori e il valore delle nuove immagini.
  • Invert. (Inverti) Inverte i colori delle immagini.
  • Make Opaque. (Rendi opaco) Rimuove completamente la trasparenza dalle immagini, rendendole opache.
  • Erase a Color. (Cancella un colore) Puoi scegliere il colore cliccando sulla casella del colore o nell'immagine di sinistra e il margine (vedi Colorize Partial). Tutti i pixel con un colore abbastanza vicino a quello selezionato verranno resi completamente trasparenti.
  • Smooth Edges. (Smussa i bordi) Smussa i bordi attorno alle parti opache della(delle) immagine(i) per renderle più belle a vedersi. È possibile ottenere un effetto migliore usando Blur sfocando solo le trasparenze (vedi sotto).
  • Opacity. (Opacità) Qui puoi cambiare l'opacità (trasparenza) di tutta l'immagine. Puoi impostarla relativa all'opacità corrente o assoluta. (I pixel completamente trasparenti resteranno sempre tali.)
  • Set Alpha from File. (Imposta Alpha da un file) Scegli un'immagine. La luminosità dei pixel di quell'immagine verrà usata come valore di trasparenza. Pixel neri significano pixel completamente trasparenti, quelli bianchi significano pixel opachi. Se l'immagine non ha la giusta dimensione viene allargata. Disponibile solo nella Standard Edition.
  • Fade. (Sbiadisci) Specifica un colore e un numero. I pixel della(delle) immagine(i) verranno sbiaditi verso questo colore dell'ammontare fornito.
  • Blur. (Sfoca) Sfocando le immagini i colori sono mischiati un po', rendendoli più vaghi. Puoi selezionare una sfocatura piccola, media o grande. Puoi anche scegliere se sfocare i colori e/o le trasparenze. Sfocare solo le trasparenze è un buon modo per smussare i bordi delle immagini.
  • Sharpen. (Affina) È una specie di contrario di sfocare. L'immagine verrà resa più netta, enfatizzando i bordi. Puoi applicare un effetto debole o forte. L'effetto speciale enfatizza ancora di più i bordi, generando un effetto buffo. Puoi affinare i colori e/o le trasparenze.
  • Outline. (Cornice) Crea una cornice attorno all'immagine. Ti viene chiesto il colore e lo spessore della cornice. Puoi anche piazzarla dentro l'immagine. Puoi indicare se rimuovere l'immagine stessa, lasciando solo la cornice, e se smussarla. Disponibile solo nella Standard Edition.
  • Shadow. (Ombra) Crea un'ombra dietro all'immagine. (Funziona solo se l'immagine è parzialmente trasparente.) Puoi indicare il colore e l'opacità dell'ombra. Puoi anche indicare la posizione relativa dell'ombra. Infimne puoi indicare se deve essere un'ombra debole o forte. Disponibile solo nella Standard Edition.
  • Glow. (Alone) Crea un alone colorato attorno all'immagine. Indica il colore, l'opacità e lo spessore dell'alone. Puoi anche piazzarlo dentro l'immagine. Disponibile solo nella Standard Edition.
  • Buttonize. (Effetto pulsante) Trasforma la(le) immagine(i) in un pulsante. Specifica il colore e l'opacità del pulsante. Devi anche indicare lo spessore del bordo del bottone e se deve essere smussato o no. Disponibile solo nella Standard Edition.
  • Gradient Fill. (Gradiente) Agguinge un gradiente alla(alle) immagine(i). Specifica i due colori del gradiente, l'opacità e il tipo di gradiente. L'impostazione di default prevede che l'immagine venga sostituita dal gradiente, nonostante la trasparenza venga mantenuta. Deselezionando la casella Replace (Sostituisci) il gradiente è mescolato con l'immagine. Selezionando la casella Change Transparency (Cambia trasparenza) l'Alpha dell'immagine originale verrà cambiata. Puoi creare grandi effetti. Ad esempio, prova a convertire un'immagine in bianco e nero e mescolarvi un gradiente. Disponibile solo nella Standard Edition.

Dovrai fare degli esperimenti con questi comandi per ottenere gli sprite che desideri.

Animation menu

Under the animation menu you can create new animated sprites out of the current sprite. There are many options and you should experiment a bit with them to create the effects you want. Also don't forget that you can always save an animated sprite and later add it to the current one. Also you can always add some empty images and delete unwanted ones. We will briefly go through the different possibilities.

  • Set Length. Here you can change the length of your animation. The animation is repeated enough times to create the number of frames you indicate. (Normally you want this to be a multiple of the current number of frames.)
  • Stretch. This command also changes the length of the animation. But this time, frames are duplicated or removed to get the right number. So if you increase the number of frames the animation goes slower and if you decrease the number it goes faster.
  • Reverse. Well, as you could guess this reverses the animation. So it is played backwards.
  • Add Reverse. This time the reverse sequence is added, doubling the number of frames. This is very useful for making an object go left and right, change color and return, etc. You sometimes might want to remove the double first and middle frame that occur.
  • Translation Sequence. You can create an animation in which the image slightly translates in each step. You must provide the number of frames and the total amount to move horizontally and vertically.
  • Rotation Sequence. Creates an animation in which the image rotates. You can either choose clockwise or counterclockwise rotation. Specify the number of frames and the total angle in degrees (360 is a complete turn). (You might need to resize the canvas first to make sure the total image remains visible during the rotation.)
  • Colorize. Creates an animation that turns the image into a particular color.
  • Fade to color. Creates an animation that fades the image to a particular color.
  • Disappear. Makes the image disappear by making it more transparent in each step.
  • Shrink. Shrinks the image to nothing. You can indicate the direction.
  • Grow. Grows the image from nothing.
  • Flatten. Flattens the image to nothing in a given direction.
  • Raise. Raises the image from a given direction.
  • Overlay. Overlays the animation with another sprite or image in a file. You indicate the file and the image(s) from that file are then placed on top of the current animation. When the image from the file has a different size it is stretched.
  • Morph. Morphs the animation to an animation or image from a file. Note that morphing works best if the two animations cover the same area of the image. Otherwise, halfway certain pixels disappear and others suddenly appear.

In particular the last two commands are very powerful. For example, to blow up an object, add a number of copies and then a number of empty frames. Then overlay it with an explosion animation. (Make sure the numbers of images match.) Alternatively, morph it to the explosion. With some practice you can make great sprites.
