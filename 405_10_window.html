La Finestra

Il gioco vero e proprio avviene in una finestra. La finestra ha delle proprietà, ad esempio può avere un bordo o no, può essere a schermo intero o meno, ecc. Di solito possono essere impostate nelle Global Game Settings, ma puoi cambiarle anche durante il gioco. Per farlo esistono le seguenti funzioni:

window_set_visible(visible) Imposta se la finestra del gioco è visibile (true) o no (false). Ovviamente di solito vorrai che la finestra rimanga visibile per tutto il gioco. Il programma non riceverà gli input da tastiera mentre la finestra è invisibile.
window_get_visible() Restitiscetrue se la finestra di gioco è visibile.
window_set_fullscreen(full) Imposta se la finestra è mostrata a schermo intero (true) o no.
window_get_fullscreen() Restituisce true se la finestra è mostrata a schermo intero.
window_set_showborder(show) Imposta se viene mostrato il bordo della finestra (true) o no. (A schermo intero non viene mai mostrato.)
window_get_showborder() Restituisce true se viene mostrato il bordo in modalità finestra.
window_set_showicons(show) Imposta se le icone Riduci a icona, Ingrandisci e Chiudi sono mostrate (true) o no. (A schermo intero non vengono mai mostrate.)
window_get_showicons() Restituisce true se sono mostrate le icone Riduci a icona, Ingrandisci e Chiudi.
window_set_stayontop(stay) Imposta se la finestra deve essere sempre in cima (true) o no.
window_get_stayontop() Restituisce true se la finestra sta sempre in cima.
window_set_sizeable(sizeable) Imposta se la finestra è dimensionabile dal giocatore (true) o no. (Il giocatore può dimensionarla solo se viene mostrato il bordo e la finestra non è a schermo intero.)
window_get_sizeable() Restituisce true se la finestra è dimensionable dal giocatore.
window_set_caption(caption) Imoosta la stringa che fa da didascalia della finestra. Di solito la specifichi mentre definisci la room e la cambi con la variabile room_caption. Quindi di solito questa funzione è inutile, a meno che tu non ti disegni la room da solo piuttosto che lasciarlo fare a GameMaker. La didascalia è visibile solo se viene mostrato il bordo e la finestra non è a schermo intero.
window_get_caption() Restituisce la didascalia della finestra.
window_set_cursor(curs) Imposta il cursore usato nella finestra. Puoi usare le seguenti costanti: (a volte sono stati usati dei simboli per rendere più efficacemente la forma dei cursori)

cr_default (normale)
cr_none (nessuno)
cr_arrow (freccia)
cr_cross (+)
cr_beam (testo)
cr_size_nesw (/ con frecce alle estremità)
cr_size_ns (| con frecce)
cr_size_nwse (\ con frecce)
cr_size_we (- con frecce)
cr_uparrow (| con freccia solo in alto)
cr_hourglass (XP: clessidra; Vista/7: cerchio che gira)
cr_drag (freccia con quadratino sotto)
cr_nodrop (simbolo divieto)
cr_hsplit (| con frecce a sinistra e a destra)
cr_vsplit (- con frecce in alto e in basso)
cr_multidrag (freccia con tanti quadratini dietro)
cr_sqlwait (clessidra con scritta "SQL")
cr_no (simbolo divieto colorato)
cr_appstart (XP: freccia e clessisra; Vista/7: freccia e cerchio che gira)
cr_help (freccia con punto interrogativo)
cr_handpoint (manina con indice puntato)
cr_size_all (+ con frecce alle estremità)

In particolare, usa cr_none per nascondere il cursore.
window_get_cursor() Restituisce il cursore usato nella finestra.
window_set_color(color) Imposta il colore di quella parte della finetra che non è usata per mostrare la room.
window_get_color() Restituisce il colore della finestra.
window_set_region_scale(scale,adaptwindow) Se la finestra è più grande della room, di solito questa viene mostrata al centro della finestra. È tuttavia possibile indicare che deve essere scalata per riempire tutta la finestra o una sua parte. Il valore 1 indica nessuna scala. Se usi il valore 0 la regione verrà scalata per coprire tutta la finestra. Se usi un valore negativo verrà espansa al massimo nella finestra mantenendo i rapporti (spesso è quello che vuoi). adaptwindow indica se la dimensione della finestra deve essere adattata (true) se la room non c'entra o no (false). Ha effetto solo se usi un fattore di scala positivo.
window_get_region_scale() Restituisce il fattore di scala usato per disegnare la regione.

The window has a position on the screen and a size. (When we talk about position and size we always mean the client part of the window without the borders.) You can change these although you hardly ever do from within your game. Normally they are determined automatically or by the player. The following functions can be used to change the window position and size. Note that these functions deal with the windowed mode. If the window is in full screen mode they can still be used but will only have effect when switching full screen mode off.

window_set_position(x,y) Sets the position of the (client part of the) window to the indicated position.
window_set_size(w,h) Sets the size of the (client part of the) window to the indicated size. Note that is the indicated size is too small to fit the drawing region it is kept large enough for the region to fit it.
window_set_rectangle(x,y,w,h) Sets the position and size of the window rectangle. (Does both previous routines in one step.)
window_center() Centers the window on the screen.
window_default() Gives the window the default size and position (centered) on the screen.
window_get_x() Returns the current x-coordinate of the window.
window_get_y() Returns the current y-coordinate of the window.
window_get_width() Returns the current width of the window.
window_get_height() Returns the current height of the window.

Again, you probably never want to use any of the window positioning functions as GameMaker takes care of these automatically.

In rare cases you might want to know the position of the mouse with respect to the window. (Normally you always use the mouse position with respect to the room or with respect to a view.) The following functions exist for this.

window_mouse_get_x() Returns the x-coordinate of the mouse in the window.
window_mouse_get_y() Returns the y-coordinate of the mouse in the window.
window_mouse_set(x,y) Sets the position of the mouse in the window to the indicated values.
